## Description

a simple SDL game

### install

#### windows

download on itch.io
[download](https://benlamouche.itch.io/mission-intersideral)

#### linux

```bash
sudo apt install libsdl2-dev libsdl2-image-dev libsdl2-mixer-dev libsdl2-ttf-dev
git clone https://github.com/idriss-code/metroidVania.git
cd metroidVania
make
./bin/main
```
